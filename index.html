Welcome to the demo of my clock !

Switch Modes:

This is a simple website show casing the features of the clock I designed during my partipation during the Hardware Product Sprint program by Google.
Below is a short explaination of the features!
Fun fact: This website was originally created to test out the display features for my clock before wiring everything up and working it out in C++.

  • Binary Mode
    • This shows the current seconds in a binary fashion!
      Seconds (0-59) are shown as a 6 bit number !
  • Decimal Mode
    • This shows the current seconds in a nicely formated way on the 8x8 display!
      Much props to the Arduino community for getting this style out there :)
  • Hour Minutes Seconds Mode
    • This shows the current time in a binary fashion!
      - Hours (1 - 12) are shown as a 4 bit number on the top row
      - Minutes (0-59) are shown as a 6 bit number on the middle row
      - Seconds (0-59) are shown as a 6 bit number on the bottom row

This is the front of my clock. No ICs are inserted for safety when bringing up!
Below is the list of components on this clock!
  • ICs
    • ATMEGA328-PU
    • MAX7219 x2
    • Real Time Clock IC (DS1307 DIP-8)
  • Power:
    • Power Jack input
    • Linear Voltage regulator (L7805 TO92)
    • 3V Battary (for RTC)
  • Crystals
    • 16MHz
    • 32.768 kHz
  • Compoents:
    • Rotary Encoder
    • LCD Module
    • 8x8 Dot Matrix Display x2
